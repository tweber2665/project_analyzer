# Project Analyzer Suite - Requirements File
# ==========================================
# 
# Analyzes project codebases to generate comprehensive reports on:
# - File structure and organization
# - Function definitions and usage
# - Variable naming and categorization
# - File paths and API endpoints
#
# Author: Tim Weber
# Python Version: >= 3.9 (required for ast.unparse in functions analyzer)

# Core Dependencies
# =================
# This project uses ONLY Python standard library modules.
# No external packages need to be installed!

# The following standard library modules are used:
# - ast          : Abstract Syntax Tree parsing for Python code analysis
# - re           : Regular expressions for pattern matching
# - os           : Operating system interface
# - sys          : System-specific parameters
# - json         : JSON encoder and decoder
# - csv          : CSV file reading and writing
# - pathlib      : Object-oriented filesystem paths
# - collections  : Specialized container datatypes (defaultdict)
# - datetime     : Date and time handling
# - time         : Time-related functions
# - argparse     : Command-line argument parsing
# - typing       : Type hints support
# - fnmatch      : Unix filename pattern matching
# - urllib.parse : URL parsing
# - mimetypes    : Map filenames to MIME types
# - subprocess   : Subprocess management
